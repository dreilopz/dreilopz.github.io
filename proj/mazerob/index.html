mazerob¶

Maze solver robot - IN DEVELOPMENT¶

Project’s website:
 http://www.dreilopz.com/mazerob
github repo:https://github.com/lopezpdvn/mazerob

Project mazerob is a robot that autonomously solves a maze. The system runs on Java Virtual Machines on a personal computer and a Lego NXT Brick configured as a master-slave architecture. Project mazerob was supervised by Professor Elisa Schaeffer and partially funded by UANL through the PROVERICYT. Project is in development.
